Requirements:

All your files will be interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.9)
Your files will be executed with numpy (version 1.25.2)
All your files must end with a new line
The first line of all your files should be exactly #!/usr/bin/env python3
Your code must use the pycodestyle style (version 2.11.1)
All your modules must have documentation (python3 -c 'print(__import__("my_module").__doc__)')
All your classes must have documentation (python3 -c 'print(__import__("my_module").MyClass.__doc__)')
All your functions (inside and outside a class) must have documentation (python3 -c 'print(__import__("my_module").my_function.__doc__)' and python3 -c 'print(__import__("my_module").MyClass.my_function.__doc__)')
Unless otherwise noted, you are not allowed to import any module except "import numpy as np"
Unless otherwise noted, you are not allowed to use any loops (for, while, etc.)
The length of your files will be tested using wc
